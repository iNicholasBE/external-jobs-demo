How it works (External Jobs)

  1. You submit a prompt — JobRunr creates an External Job
  2. The job trigger calls the Replicate API to start a GPU prediction
  3. The job enters PROCESSED state, waiting for the GPU to finish
  4. A poller detects completion and signals the External Job as succeeded
